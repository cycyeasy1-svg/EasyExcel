# Change Log

EasyExcel の変更履歴です。形式は [Keep a Changelog](https://keepachangelog.com/ja/1.1.0/) に、バージョニングは [Semantic Versioning](https://semver.org/lang/ja/) に準拠します。

## [1.0.0] - 2026-07-07

初回リリース。

### Added

- `.xlsx` / `.xlsm` / `.xls` / `.csv` / `.tsv` / `.ods` のプレビューと編集
- Univer ベースのスプレッドシートエンジン(複数シート、スタイル、結合セル、数式、ハイパーリンク、入力規則、シート保護情報、画像、条件付き書式の表示)
- 検索と置換
- `.xlsx` / `.xlsm` の上書き保存における XML 差分パッチ方式のバイトセーフ保存 — 編集していないパート(図形・グラフ・ピボットテーブル・マクロ・印刷設定など)を 1 バイトも変更しない
- バイトセーフ保証のための構造変更操作の無効化(行・列・シートの挿入/削除、結合、行高・列幅、固定、入力規則・条件付き書式の編集)
- 保存 / 名前を付けて保存ボタンと保存完了トースト
- CSV / TSV の文字コード自動判定
