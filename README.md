# EasyExcel

EasyExcel は、Visual Studio Code の中で Excel 系ファイルをそのままプレビュー・編集できる拡張機能です。ファイルを開くだけでスプレッドシートビューが起動し、Excel を立ち上げることなく閲覧・修正・保存ができます。

## 対応形式

| 形式 | 読み込み | 保存 |
| --- | --- | --- |
| `.xlsx` | ✔ | ✔（差分パッチによるバイトセーフ保存） |
| `.xlsm` | ✔ | ✔（マクロを変更せず保持） |
| `.xls` | ✔ | `.xlsx` として保存 |
| `.csv` / `.tsv` | ✔（文字コード自動判定） | ✔ |
| `.ods` | ✔ | `.xlsx` として保存 |

## 主な機能

- ワークブック・複数シートの表示と切り替え
- セル編集、数式、ハイパーリンク
- 検索と置換
- 基本的なスタイル、結合セル、入力規則、シート保護情報、画像の表示
- 条件付き書式の表示
- 保存(`Ctrl+S`)と名前を付けて保存

## 差分だけを書き込むバイトセーフ保存(.xlsx / .xlsm)

`.xlsx` / `.xlsm` を上書き保存するとき、EasyExcel はファイル全体を再構築しません。編集したセルに対応する XML パートだけを元のファイルに継ぎ当てます。

**編集していない内容 — 図形・テキストボックス・グラフ・ピボットテーブル・マクロ・印刷設定などを含む — は 1 バイトも変わりません。** 他のツールで作り込んだブックを開いて保存しても、触っていない部分が壊れる心配がありません。

この保証を守るため、ビューア内では構造を変える操作(行・列・シートの挿入や削除、セル結合、行高・列幅の変更、ウィンドウ枠の固定、入力規則・条件付き書式の編集)は無効化しています。これらの操作が必要な場合は Excel 等で行ってください。なお「名前を付けて保存」で書き出したコピーは通常のフル書き出しとなります。

`.csv` と `.tsv` は単一シートのテキスト形式です。スタイル、結合セル、画像などのリッチな情報は保持できないため、必要に応じて `.xlsx` として保存してください。

## 使い方

1. 拡張機能をインストールします。
2. エクスプローラーで対象ファイルをクリックすると EasyExcel で開きます。
3. 既定のエディタを変更したい場合は、ファイルを右クリック →「プログラムから開く」→「既定のエディタとして設定」から切り替えられます。

## プライバシー

EasyExcel はすべての処理をローカルで行います。開いたファイルの内容を外部サーバーへ送信することはなく、テレメトリも収集しません。

## 開発

```bash
npm install
npm run build    # 本番ビルド
npm run dev      # Vite + 拡張ホストの開発ビルド
npm test         # vitest
```
