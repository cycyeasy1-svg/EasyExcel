# EasyExcel

EasyExcel 是一款在 Visual Studio Code 中直接预览和编辑 Excel 系列文件的扩展。点击文件即可打开电子表格视图，无需启动 Excel 就能完成浏览、修改和保存。

## 支持的格式

| 格式 | 打开 | 保存 |
| --- | --- | --- |
| `.xlsx` | ✔ | ✔（差分补丁字节级无损保存） |
| `.xlsm` | ✔ | ✔（宏原样保留，不做任何改动） |
| `.xls` | ✔ | 以 `.xlsx` 格式另存 |
| `.csv` / `.tsv` | ✔（自动检测编码） | ✔ |
| `.ods` | ✔ | 以 `.xlsx` 格式另存 |

## 主要功能

- 工作簿与多工作表的展示和切换
- 单元格编辑、公式、超链接
- 查找与替换
- 基础样式、合并单元格、数据验证、工作表保护信息、图片的展示
- 条件格式的展示
- 保存（`Ctrl+S`）与另存为
- 界面语言可切换（简体中文 / English，默认中文）

## 只写差分的字节级无损保存（.xlsx / .xlsm）

覆盖保存 `.xlsx` / `.xlsm` 时，EasyExcel 不会重建整个文件，只把编辑过的单元格对应的 XML 部件补丁进原文件。

**未编辑的内容——包括图形、文本框、图表、透视表、宏、打印设置等——一个字节都不会改变。** 用其他工具精心制作的工作簿，在这里打开、保存后，没动过的部分不会有任何损坏风险。

为了守住这条保证，查看器内禁用了会改变结构的操作（行/列/工作表的插入删除、合并单元格、行高列宽调整、冻结窗格、数据验证和条件格式的编辑）。需要这些操作时请在 Excel 等软件中进行。另外，通过「另存为」导出的副本走常规完整导出。

`.csv` 和 `.tsv` 是单工作表的纯文本格式，无法保留样式、合并单元格、图片等富格式信息，需要时请另存为 `.xlsx`。

## 使用方法

1. 安装本扩展。
2. 在资源管理器中点击目标文件，即以 EasyExcel 打开。
3. 如需更改默认编辑器，可右键文件 →「打开方式」→「配置默认编辑器」进行切换。

## 隐私

EasyExcel 的全部处理都在本地完成，不会把打开的文件内容发送到任何外部服务器，也不收集遥测数据。

## 开发

```bash
npm install
npm run build    # 生产构建
npm run dev      # Vite + 扩展宿主的开发构建
npm test         # vitest
```
