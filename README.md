# EasyExcel

EasyExcel は、デスクトップ版 Visual Studio Code で Excel 系ファイルをプレビュー・編集するための拡張機能です。

## 対応形式

- `.xlsx`
- `.xlsm`
- `.xls`
- `.csv`
- `.tsv`
- `.ods`

## 主な機能

- ワークブックとシートの表示
- セル編集
- 検索と置換
- 保存と名前を付けて保存
- 基本的なスタイル、結合セル、数式、ハイパーリンク、入力規則、保護情報、画像の読み書き

### 差分だけを書き込む保存（.xlsx / .xlsm）

`.xlsx` / `.xlsm` を上書き保存するとき、EasyExcel はファイルを再構築せず、編集したセルの XML だけを元のファイルに継ぎ当てします。**編集していない内容 — 図形・テキストボックス・グラフ・ピボットテーブル・マクロ・印刷設定などを含む — は 1 バイトも変わりません。**

この保証のため、ビューア内では構造を変える操作（行・列・シートの挿入や削除、結合、行高・列幅、ウィンドウ枠の固定、入力規則、条件付き書式の編集）は無効化しています。これらの操作は Excel で行ってください（「名前を付けて保存」で書き出したコピーには適用されません）。

`.csv` と `.tsv` は単一シートのテキスト形式です。スタイル、結合セル、画像などのリッチな情報は保持できないため、必要に応じて `.xlsx` として保存してください。

## 開発

```bash
npm install
npm run build
```

開発時は次のコマンドで Vite と拡張ホストのビルドを起動します。

```bash
npm run dev
```
